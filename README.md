# PocketIQ

A personal finance application designed to help students and young professionals manage their money.

PocketIQ is a Python project that I am building from scratch as part of my journey in learning Python, software development, Git, and AI.

## Current Features (Version 2)

- Income tracking
- Expense tracking
- Expense categories
- Edit and delete expenses
- Spending statistics
  - Average expense
  - Largest expense
  - Smallest expense
- Save and reload data using JSON
- Version control using Git and GitHub

## Technologies Used

- Python
- JSON
- Git
- GitHub
